# 快速启动指南

## 后端启动

1. **进入后端目录**
```bash
cd zhuangxiu-backend
```

2. **安装依赖**
```bash
npm install
```

3. **配置环境变量**
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑 .env 文件，确保 JWT_SECRET 已设置
# 可以使用默认值，但生产环境请修改为强密码
```

4. **启动后端服务**
```bash
# 开发模式（自动重启）
npm run dev

# 或生产模式
npm start
```

后端服务将在 `http://localhost:5001` 启动

## 前端启动

1. **进入前端目录**
```bash
cd zhuangxiu-frontend
```

2. **安装依赖**
```bash
npm install
```

3. **启动前端服务**
```bash
npm run dev
```

前端服务将在 `http://localhost:5173` 或类似端口启动（Vite默认端口）

## 测试登录注册

1. 访问前端地址（如 `http://localhost:5173`）
2. 点击"立即注册"创建账号
3. 注册成功后会自动跳转到登录页
4. 使用注册的账号密码登录

## 常见问题

### 1. 后端启动失败：JWT_SECRET 未设置
**解决方案**：确保 `zhuangxiu-backend/.env` 文件存在且包含 `JWT_SECRET` 配置

### 2. 前端无法连接后端
**解决方案**：
- 检查后端是否正常启动（访问 `http://localhost:5001/health`）
- 检查 `zhuangxiu-frontend/src/services/api.js` 中的 `baseURL` 是否正确

### 3. 数据库错误
**解决方案**：
- 确保 `zhuangxiu-backend` 目录有写入权限
- 删除 `db.sqlite` 文件后重新启动（会重新创建数据库）

### 4. 文件上传失败
**解决方案**：
- 确保 `zhuangxiu-backend/uploads` 目录存在
- 确保该目录有写入权限
